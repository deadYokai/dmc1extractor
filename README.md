# DMC1 Resource Repacker

## Depency:
* Python 3

## Usage:
* ### Dat/Itm Files:
    ```bash
    $ python3 datExtractor.py <datfile>
    ```

* ### Fsd Files:
    ```bash
    $ python3 fsdExtractor.py <fsdfile>
    ```

* ### Texture Files:

    #### Unpack:
    ```bash
    $ python3 textureRepack.py <texturefile>
    ```
    #### Repack:
    ```bash
    $ python3 textureRepack.py -p <texturefile>
    ```

    #### Supported texture files:
    - .tm2
    - .ip2
    - .t32
